Welcome to TIC TAC TRIVIA!

  1. To get started, enter a username on the next page.

  2. If no other players are online, you can invite a friend.

  3. Players can discuss what category and difficulty to select by using a chat feature.

  4. The game will begin after a category is selected.

  5. Turns will alernate between players. Player 1 has the first turn to select a square on the board.

  6. Both players will be given 20 seconds to answer the question.

  7. The first player to correctly answer the question will be granted the selected square.

  8. If time runs out or if no one anwers the question correctly, the next player gets to select a square.

  9. The first player with 3 squares in a row wins the game!

  10. Have Fun!
