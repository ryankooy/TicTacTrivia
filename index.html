  1. To get started enter a username on the next page.

  2. If no other players are online, you can invite a friend.

  3. Once Player 1 and Player 2 are selected, the game will start and other players must wait until the game is complete in order to start a new game.

  4. Players can discuss what category and difficulty to select using a chat feature.

  5. When these have been selected the game will begin.

  6. Player 1 has the first turn to select a square on the board

  7. Both players will be given 20 seconds to answer the question.

  8. The first player to correctly answer the question will be granted the selected square.

  9. Then player 2 will have the opportunity to select a square.

  10. If time runs out or if no one anwers the question correctly, the next player gets to select a square.

  11. The first player with 3 squares in a row wins the game!
